### Integration Issues

**AI-02 — isAI field is dead end (Low severity)**
- `proxy/server.js` computes `isAI` and marks it `required` in the tweetSchema
- `ProcessedTweetResult` in `types.ts` declares `isAI?: boolean`
- `claudeService.ts` fallback omits `isAI` entirely (schema non-conformant)
- `App.tsx` never reads or stores `isAI` — the field is silently discarded
- Fix: either wire `isAI` to UI/storage (display "AI-processed" badge) or remove from proxy schema

**Phase 4 incomplete — `src/index.html` importmap (Low severity)**
- File: `src/index.html` line 19 still has `"@google/genai": "https://aistudiocdn.com/..."` in the ES module importmap
- Phase 4 deleted `geminiService.ts` and removed from `package.json` but did not check `index.html`
- Non-breaking (no code imports `@google/genai`) but causes a redundant CDN fetch on every page load
